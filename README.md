# Women Safety Application – Android Platform

## Introduction
Women Safety Application is an Android-based mobile application designed to help enhance personal safety for women. The application allows users to quickly send emergency alerts, share their real-time location, and contact trusted people in dangerous situations.

## Technologies Used
- Android Studio
- Java
- Firebase Realtime Database
- Google Maps API
- XML Layout
- Git & GitHub

## Features
- User registration and login
- Emergency SOS button
- Real-time location sharing
- Send alert messages to emergency contacts
- GPS tracking using Google Maps
- Add and manage trusted contacts
- Location history tracking
- Notification system for emergency alerts
- Simple and user-friendly interface

## System Requirements
- Android 6.0 or higher
- Internet connection
- GPS enabled

## Installation
1. Clone the repository
2. Open the project using Android Studio
3. Sync Gradle dependencies
4. Run the application on an Android device or emulator

## Future Improvements

- Voice activation for SOS
